dokipen's PEAR Channel Server

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover dokipen.github.com/pear
Listing available packages: 
pear remote-list -c dokipen
Installing a package: 
pear install dokipen/package_name
Installing a specific version/stability: 
pear install dokipen/package_name-1.0.0
pear install dokipen/package_name-beta
Receiving updates via a feed: 
http://dokipen.github.com/pear/feed.xml
